🎮 Tic Tac Toe - Frontend Project

A simple and interactive browser-based Tic Tac Toe game.
Beginner friendly

🚀 Tech Stack 🚀
- HTML
- CSS
- JS

⭐ Features:
- Win and draw detection
- Visual win highlight
- Reset Game option
- Display name of winner or draw
